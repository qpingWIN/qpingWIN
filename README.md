## Projects

### [SQL in 7 Days](https://github.com/qpingWIN/sql-7-days)
7-day sprint covering SQL foundations (SELECT, filtering, sorting), aggregations with GROUP BY/HAVING, JOINs, subqueries/CTEs/set operations, window functions, date and string functions with CASE and NULL handling, and database design (indexes, transactions, query optimization).

### [Bible RAG Chatbot](https://github.com/qpingWIN/bible-rag-chatbot)
Local retrieval-augmented generation pipeline over 367k passages — KJV, BSB, and Matthew Henry's Complete Commentary. Built with sentence-transformers, FAISS, and llama3.2 via Ollama. No API, runs fully offline.

### [rPPG Heart Rate Estimation](https://github.com/qpingWIN/rPPG-mobile-augmentation)
MSc dissertation at UCL in collaboration with Klarity Health. Trained TS-CAN, DeepPhys, and EfficientPhys on UBFC-rPPG; tested on PURE dataset. Face-box crop augmentation improved Pearson correlation from 0.67 → 0.96.

### [Hate Speech Detection](https://github.com/qpingWIN/Hate_speech_detection)
BiLSTM classifier on 24,783 tweets. Addressed class imbalance via stratified sampling and threshold tuning. 86% validation accuracy across 3-class setup (hate / offensive / neutral).

### [House Price Prediction](https://github.com/qpingWIN/House_Price_Prediction)
Regression benchmark on the Ames Housing dataset. Compared SVR, Linear Regression, and Random Forest. RF achieved R² = 0.75, MAPE = 13.4% after feature engineering on 80 predictors.

### [Wikipedia News Scraper](https://github.com/qpingWIN/Wikipedia_scraping)
Python scraper targeting Wikipedia's current events section. Extracts and structures breaking news headlines with timestamps, outputs to a clean tabular format for downstream analysis.
